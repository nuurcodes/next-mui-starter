# DApp Frontend Starter

A DApp Frontend starter project using NextJS

## Getting Started

### Dependencies

* node v10.13+
* npm
* yarn

### Installing dependencies

```
yarn
```

### Running the app

* Development
```
yarn dev
```

* Production
```
yarn build
yarn start
```
